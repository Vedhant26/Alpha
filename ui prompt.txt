STEP 6 — 3D PARALLAX SCROLLING
Implement on the board view and all scrollable views:
javascript// On mousemove over the board
document.addEventListener('mousemove', (e) => {
  const x = (e.clientX / window.innerWidth - 0.5)  // -0.5 to 0.5
  const y = (e.clientY / window.innerHeight - 0.5)

  // Cloud layers shift slightly opposite to mouse
  cloudLayer1.style.transform = `translateX(${x * -12}px) translateY(${y * -6}px)`
  cloudLayer2.style.transform = `translateX(${x * -20}px) translateY(${y * -10}px)`
  cloudLayer3.style.transform = `translateX(${x * -8}px) translateY(${y * -4}px)`

  // Cards tilt very slightly toward mouse (3D feel)
  // Apply to hovered card only
  hoveredCard.style.transform = `
    perspective(1000px)
    rotateX(${y * -4}deg)
    rotateY(${x * 4}deg)
    translateZ(8px)
  `
})
Parallax must use Easing-Gentle with Duration-Slow so it lags slightly behind the mouse — that lag IS the 3D depth illusion. Never snap instantly.
